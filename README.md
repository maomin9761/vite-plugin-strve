# vite-plugin-strve

Strve loader plugin for vite.

## Install

```bash
npm i vite-plugin-strve -D
```

## Usage

```js
import { defineConfig } from 'vite';
import { strve } from 'vite-plugin-strve';

export default defineConfig({
	plugins: [strve()],
});
```
